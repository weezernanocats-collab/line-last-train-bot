# 株式ポートフォリオ管理スプレッドシート（修正版）

## 変更点

- **EPS/BPSは手動入力**に変更（スクレイピング不安定のため）
- **目標株価を追加**（アナリスト予想の代替指標）
- **乖離率を追加**（目標株価と現在値の差）
- 現在値は**IMPORTXML関数**で取得（GASカスタム関数より安定）

## セットアップ

### 1. スプレッドシート作成

1. Google ドライブで「新規」→「Google スプレッドシート」
2. 「拡張機能」→「Apps Script」を開く
3. `StockPortfolio.gs` の内容をコピー＆ペースト
4. 保存してスプレッドシートに戻る
5. ページを再読み込み（F5）
6. メニュー「株式管理」→「初期化（シート作成）」を実行

### 2. 銘柄マスターを設定

| 列 | 項目 | 入力内容 |
|---|---|---|
| A | コード | 4桁の証券コード |
| B | 種別 | 業種（設定シートと一致させる） |
| C | 銘柄名 | 会社名 |
| D | EPS | 1株当たり利益（手動入力） |
| E | BPS | 1株当たり純資産（手動入力） |
| F | 目標株価 | アナリストコンセンサス（手動入力） |
| G | 更新日 | 最終更新日 |

### 3. ポートフォリオで管理

コード、数量、取得単価を入力すると、他の項目が自動計算されます。

## シート構成

### ポートフォリオ（メインシート）

| 列 | 項目 | 入力方法 |
|---|---|---|
| A | コード | 手動入力 |
| B | 種別 | 自動（マスターから） |
| C | 銘柄 | 自動（マスターから） |
| D | 数量 | 手動入力 |
| E | 取得単価 | 手動入力 |
| F | 買い | 自動判定 |
| G | 理論株価(PER) | 自動（適正PER × EPS） |
| H | 理論株価(PBR) | 自動（適正PBR × BPS） |
| I | 目標株価 | 自動（マスターから） |
| J | 乖離率 | 自動（目標株価との差） |
| K | 現在値 | 自動（IMPORTXML） |
| L | 取得額 | 自動（取得単価 × 数量） |
| M | 評価額 | 自動（現在値 × 数量） |
| N | 損益(円) | 自動（評価額 - 取得額） |
| O | 損益(%) | 自動（損益 / 取得額） |

### 設定シート

業種別の適正PER/PBRを設定します。

| 種別 | 適正PER | 適正PBR | 備考 |
|---|---|---|---|
| 自動車 | 10 | 0.8 | 景気敏感 |
| 電気機器 | 18 | 2.0 | 成長期待 |
| 銀行 | 8 | 0.5 | 低PBR傾向 |
| 通信 | 12 | 1.2 | 安定配当 |

## 理論株価の計算式

### PER基準
```
理論株価 = 適正PER × EPS
```
- **適正PER**: 設定シートで業種別に設定
- **EPS**: 銘柄マスターで手動入力

### PBR基準
```
理論株価 = 適正PBR × BPS
```
- **適正PBR**: 設定シートで業種別に設定
- **BPS**: 銘柄マスターで手動入力

## 買いシグナル

「買い」列に「○」が表示される条件：
- 理論株価（PER基準）> 現在値、または
- 理論株価（PBR基準）> 現在値、または
- 目標株価 > 現在値

いずれかを満たせば「割安」と判断して「○」表示されます。

## アナリスト予想の代替指標

12ヶ月後のPER/PBR予想は無料での自動取得が困難なため、以下を代替として使用しています：

| 代替指標 | 説明 |
|---|---|
| **目標株価** | アナリストコンセンサスの目標株価 |
| **乖離率** | (目標株価 - 現在値) / 現在値 |

### 確認先

- **みんかぶ**: https://minkabu.jp/stock/[コード]
  - 「目標株価」「アナリスト予想」で確認
- **Yahoo!ファイナンス**: 一部銘柄で予想あり
- **会社四季報オンライン**: EPS/BPS確認
- **マネックス証券 銘柄スカウター**: 詳細な予想データ

## EPS/BPSの確認方法

### みんかぶで確認

1. https://minkabu.jp/stock/[コード] にアクセス
2. 「業績・財務」タブをクリック
3. 「1株益（EPS）」「1株純資産（BPS）」を確認

### 会社四季報で確認

1. 四季報オンラインまたは書籍版を確認
2. 「1株利益」「1株純資産」の項目

## 注意事項

- **現在値の取得**: Google Financeから取得（15-20分遅延あり）
- **IMPORTXML制限**: 多数の銘柄（20以上）で取得が不安定になる場合あり
- **EPS/BPS更新**: 決算発表後に手動更新が必要

## トラブルシューティング

### 現在値が「取得中...」のまま

- 証券コードが正しいか確認
- Google Financeで当該銘柄が存在するか確認
- 時間をおいて再度確認（IMPORTXML制限）

### 理論株価が表示されない

- 銘柄マスターにEPS/BPSが入力されているか確認
- 設定シートに種別が登録されているか確認

### 種別・銘柄名が表示されない

- 銘柄マスターにコードが登録されているか確認
- コードの形式を確認（数値のみ、4桁）
