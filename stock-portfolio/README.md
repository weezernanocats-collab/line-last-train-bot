# 株式ポートフォリオ管理スプレッドシート（v3）

## 主な機能

- **理論株価の計算**（PER基準・PBR基準）
- **実績PER/PBR**の自動計算
- **配当利回り・ROE**の表示
- **割安シグナル**の自動判定
- 現在株価の自動取得（Google Finance）

## セットアップ

1. Google ドライブで「新規」→「Google スプレッドシート」
2. 「拡張機能」→「Apps Script」を開く
3. `StockPortfolio.gs` の内容をコピー＆ペースト
4. 保存してスプレッドシートを再読み込み
5. メニュー「株式管理」→「初期化（シート作成）」

## シート構成

### 銘柄マスター

保有銘柄の基本情報を登録します。

| 列 | 項目 | 単位 | 説明 |
|---|---|---|---|
| A | コード | - | 4桁の証券コード |
| B | 種別 | - | 業種（設定シートと一致させる） |
| C | 銘柄名 | - | 会社名 |
| D | EPS | 円 | 1株当たり利益（例: 280） |
| E | BPS | 円 | 1株当たり純資産（例: 3200） |
| F | 配当 | 円 | 年間配当金（例: 75） |
| G | ROE | % | 自己資本利益率（例: 10.5） |
| H | 目標株価 | 円 | アナリストコンセンサス |
| I | 更新日 | - | 最終更新日 |

**重要**: EPS/BPSは**円単位**で入力してください。

### ポートフォリオ

| 列 | 項目 | 入力 | 説明 |
|---|---|---|---|
| A | コード | 手動 | 証券コード |
| B | 種別 | 自動 | マスターから取得 |
| C | 銘柄 | 自動 | マスターから取得 |
| D | 数量 | 手動 | 保有株数 |
| E | 取得単価 | 手動 | 購入時の株価 |
| F | 現在値 | 自動 | Google Financeから |
| G | 買い | 自動 | 割安シグナル（○） |
| H | 理論株価(PER) | 自動 | 適正PER × EPS |
| I | 理論株価(PBR) | 自動 | 適正PBR × BPS |
| J | 目標株価 | 自動 | マスターから |
| K | 実績PER | 自動 | 現在値 ÷ EPS |
| L | 実績PBR | 自動 | 現在値 ÷ BPS |
| M | 配当利回り | 自動 | 配当 ÷ 現在値 |
| N | ROE | 自動 | マスターから |
| O | 取得額 | 自動 | 取得単価 × 数量 |
| P | 評価額 | 自動 | 現在値 × 数量 |
| Q | 損益(円) | 自動 | 評価額 - 取得額 |
| R | 損益(%) | 自動 | 損益 ÷ 取得額 |

### 設定シート

業種別の適正PER/PBRを設定します。

| 種別 | 適正PER | 適正PBR | 備考 |
|---|---|---|---|
| 自動車 | 10 | 0.9 | 景気敏感 |
| 電気機器 | 18 | 2.0 | 成長期待 |
| 銀行 | 8 | 0.5 | 低PBR傾向 |
| 通信 | 12 | 1.2 | 安定配当 |

## 投資指標の見方

### 実績PER（株価収益率）

```
実績PER = 現在値 ÷ EPS
```

- **低いほど割安**（同業種比較で）
- 一般に PER 12以下 → 割安（緑）
- 一般に PER 25以上 → 割高（赤）

### 実績PBR（株価純資産倍率）

```
実績PBR = 現在値 ÷ BPS
```

- **低いほど割安**
- PBR 1以下 → 解散価値以下＝割安（緑）
- PBR 3以上 → 割高傾向（赤）

### 配当利回り

```
配当利回り = 年間配当 ÷ 現在値
```

- 3%以上 → 高配当（緑）
- インカムゲイン重視の投資家に重要

### ROE（自己資本利益率）

```
ROE = EPS ÷ BPS × 100
```

- **高いほど効率的に利益を生み出す企業**
- 10%以上 → 優良企業（緑）
- ウォーレン・バフェットの投資基準の一つ

### 理論株価

```
理論株価(PER) = 適正PER × EPS
理論株価(PBR) = 適正PBR × BPS
```

業種別の適正PER/PBRを設定シートで管理し、それに基づいた理論上の株価を計算します。

## 割安シグナル（買い）

「○」が表示される条件（いずれかを満たす場合）：

1. **実績PER < 業種適正PER** → 収益面で割安
2. **実績PBR < 業種適正PBR** → 資産面で割安
3. **目標株価 > 現在値** → アナリスト予想で上昇余地あり

## EPS/BPSの確認方法

### みんかぶで確認

1. `minkabu.jp/stock/[コード]` にアクセス
2. 「業績・財務」タブをクリック
3. 確認項目：
   - 1株益（EPS）
   - 1株純資産（BPS）
   - 1株配当
   - ROE

### Yahoo!ファイナンスで確認

1. `finance.yahoo.co.jp/quote/[コード].T` にアクセス
2. 「企業情報」→「財務」で確認

### 会社四季報

書籍版またはオンラインで「1株利益」「1株純資産」を確認

## 条件付き書式（色分け）

| 指標 | 緑（良好） | 赤（注意） |
|---|---|---|
| 実績PER | 12以下 | 25以上 |
| 実績PBR | 1以下 | 3以上 |
| 配当利回り | 3%以上 | - |
| ROE | 10%以上 | - |
| 損益 | プラス | マイナス |
| 買いシグナル | ○ | - |

## 注意事項

- **現在値**: Google Financeから取得（15-20分遅延あり）
- **EPS/BPS**: 決算発表後に手動更新が必要
- **IMPORTXML制限**: 銘柄数が多い（20以上）と不安定になる場合あり

## トラブルシューティング

### 理論株価が異常に大きい

→ BPSの入力値を確認してください。
- 正しい例: `3200`（円単位）
- 間違い例: `3200000000`（純資産総額を入力している）

### 現在値が「...」のまま

→ しばらく待つか、証券コードを確認してください。

### 種別・銘柄が表示されない

→ 銘柄マスターにコードが登録されているか確認してください。
